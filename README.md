## Installation

    Clone the repo git clone https://github.com/Wratten/E-Commerce-Back-End.git Install dependencies npm install

## Usage

To run this app,

    npm run seed

    node server.js

The API endpoints can be tested using Insomnia, Postman or another route testing application.
